# LyriSoul 用户指南 🎵 —— 赋予音乐视觉灵魂

欢迎使用 **LyriSoul**！这是一个为您提供极致歌词同步体验的桌面应用。本指南将帮助您从零开始配置并享受您的音乐之旅。

---

## 1. 快速上手：准备您的 Spotify 凭据 (BYOK)

由于 Spotify 的政策限制，每位用户需要拥有自己的“通行证”（Client ID）来连接音乐数据。这只需花费您 1 分钟时间，且完全免费。

### 步骤 A：前往控制台
访问 [Spotify Developer Dashboard](https://developer.spotify.com/dashboard) 并使用您的 Spotify 账号登录。

### 步骤 B：创建应用
1. 点击右上角的 **“Create App”**。
2. **App name**: 填入 `LyriSoul`。
3. **App description**: 填入 `My custom lyrics player`。
4. **Redirect URI**: **重要！** 请务必填入：`http://127.0.0.1:666/callback`。
5. 勾选同意条款并点击 **“Save”**。

### 步骤 C：获取 Client ID
1. 在应用详情页点击 **“Settings”**。
2. 您会看到 **Client ID**。
3. **妥善保存这串字符**，您在第一次启动 LyriSoul 时需要用到它。

---

## 2. 安装与启动

1. **下载程序**：运行您获取到的 `LyriSoul_Commercial.exe`（或 `LyriSoul.exe`）。
2. **初始化配置**：
   - 首次启动时，程序会弹出一个美观的配置界面。
   - 贴入您在第一步中获取的 **Client ID**。
   - 点击 **“保存 Client ID”**。
3. **完成授权**：
   - 您的默认浏览器会自动打开一个 Spotify 授权页面。
   - 点击 **“Agree”**（同意）。
4. **开启音乐**：回到 LyriSoul 窗口，开始在 Spotify 播放歌曲，歌词将自动同步！

---

## 3. 核心功能说明

### 🛸 桌面交互
- **无压力拖拽**：点击窗口任何非按钮区域均可自由拖动。
- **流畅缩放**：将鼠标移至窗口边缘或角落，即可丝滑调整窗口大小。
- **系统托盘**：关闭窗口后，LyriSoul 会常驻右下角托盘，右键点击可重新唤醒或彻底退出。

### 🎤 歌词同步与控制
- **交互排版**：点击任何一行歌词，Spotify 播放器将自动跳转到该时间点。
- **多级匹配**：若当前曲目歌词不准，系统会自动尝试从多个数据源抓取。
- **进度条**：支持在底部的进度条上点击或拖拽，实现精准进度控制。

### 🌍 多语言支持
- 点击右上角的 **语言切换按钮**（图标为 `EN` 或 `文`），可在中英文界面间即时切换。

## 4. 常见问题 (FAQ)

### Q: 为什么点击登录后，浏览器显示“无法连接”？
**A**: 请检查您在 Spotify Dashboard 中设置的 **Redirect URI** 是否完全等于 `http://127.0.0.1:666/callback`。如果多了一个空格或字符，授权将失败。

### Q: 为什么歌词一直显示“Searching...”？
**A**: 
1. 确认您正在播放歌曲。
2. 系统正在尝试从多个源匹配。如果某些极其冷门的曲目或纯音乐无法找到歌词，请稍候或尝试切换下一曲。

### Q: 窗口顶部的“黑边”是什么？
**A**: 如果您在使用较早版本的 Windows 或极少数显卡驱动下看到黑边，请尝试重新编译或运行我们提供的 **Commercial 版**，该版本针对 Win11 圆角和透明度进行了深度优化。

### Q: 我想更换一套凭据（Client ID），该怎么做？
**A**: 点击登录界面的齿轮图标（设置），即可重新输入 Client ID。
